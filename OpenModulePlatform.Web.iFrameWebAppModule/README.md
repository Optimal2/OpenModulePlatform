# OpenModulePlatform.Web.iFrameWebAppModule

Proof-of-concept Razor module for OpenModulePlatform that renders the shared OMP top bar, a module-local URL selector in the module header, and an iframe whose source is resolved from `omp_iframe_module.urls` through a named URL set in `omp_iframe_module.url_sets` / `omp_iframe_module.url_set_urls`.
