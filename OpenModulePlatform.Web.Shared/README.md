# OpenModulePlatform.Web.Shared

Shared library for OMP web applications.

Included building blocks:

- `AddOmpWebDefaults(...)`
- `UseOmpWebDefaults(...)`
- `SqlConnectionFactory`
- `RbacService`
- `OmpPageModel`
- `OmpSecurePageModel`
- `HttpRequest.GetPublicBaseUrl()`
